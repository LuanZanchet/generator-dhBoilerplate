## Install
```
$ npm install -g generator-dhboilerplate
```


## Usage
Jump in your Working Directory and type: 

```
yo dhBoilerplate
```


## Commands
Intialize your Project

```
gulp init
```

Default Task with BrowserSync (CSS and HTML Injection)
```
gulp
```
